# Amazon Data Cleaning

## Project Overview
This project focuses on cleaning and preprocessing the Amazon sales dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Tasks Performed
- Removed duplicates
- Handled missing values
- Cleaned product data
- Standardized columns
- Exported cleaned dataset
