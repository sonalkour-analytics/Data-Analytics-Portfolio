# HR Analytics Data Cleaning

## Project Overview
This project focuses on cleaning and preprocessing the HR Analytics dataset using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Tasks Performed
- Removed duplicate records
- Handled missing values
- Cleaned text columns
- Standardized data
- Exported cleaned dataset
