## 📌 Project Overview
This project analyzes Uber trip data to identify business insights using Power BI and Python.

## 🛠 Tools Used
- Python (Pandas)
- Power BI
- Excel

## 📊 KPIs
- Total Trips
- Total Revenue
- Average Fare
- Average Trip Distance
- Payment Methods
- Peak Hours
- Monthly Trends

## 📈 Dashboard Features
- KPI Cards
- Bar Charts
- Line Charts
- Slicers
- Interactive Dashboard

## 📂 Dataset
Uber Trip Dataset (Cleaned using Python)
